Stack Overflow Viewer Application

This application is an evaluated project for our 1st master year.


The StackOverflow Viewer Application is an SPA as a proof of concept for our developer portfolio.
This project is composed of a MySQL Datbase, linked to a C# backend trough the Entity framework.
We use the ASP.NET framework to serve a RESTFUL api.

The front-end is managed by the Knockout.js framework using the MVVM architectural pattern.
